# MOEX Data Analysis Project

Проект для анализа данных Московской биржи (MOEX) с фокусом на акции Газпрома.

## Описание

Этот проект позволяет:
- Получать данные о ценных бумагах с MOEX API
- Загружать исторические данные по акциям
- Строить графики динамики цен
- Анализировать рыночные данные

## Файлы проекта

- `test_moex_3.py` - получение информации о ценных бумагах
- `extraction_all_data.py` - загрузка исторических данных
- `plot_gazp_price.py` - построение графиков цен

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <URL_репозитория>
cd Markovic_Black-Letterman
```

### 2. Создание виртуального окружения
```bash
python -m venv .venv
```

### 3. Активация виртуального окружения
**Windows:**
```bash
.venv\Scripts\activate
```

**Linux/Mac:**
```bash
source .venv/bin/activate
```

### 4. Установка зависимостей
```bash
pip install -r requirements.txt
```

## Использование

### Получение данных о ценных бумагах
```bash
python test_moex_3.py
```

### Загрузка исторических данных
```bash
python extraction_all_data.py
```

### Построение графика
```bash
python plot_gazp_price.py
```

## Зависимости

- requests - для работы с MOEX API
- pandas - для обработки данных
- matplotlib - для построения графиков
- apimoex - библиотека для работы с MOEX

## Примечания

- Данные загружаются с официального API MOEX
- Исторические данные сохраняются в CSV файлы
- График показывает динамику цен с отмеченными минимумом и максимумом
