# Website для MOEX Data Analysis

Веб-интерфейс для проекта анализа данных Московской биржи.

## Структура

```
website/
├── app.py              # Основное Flask приложение
├── templates/          # HTML шаблоны
│   ├── base.html      # Базовый шаблон
│   ├── index.html     # Главная страница
│   ├── register.html  # Форма регистрации
│   ├── login.html     # Форма входа
│   └── dashboard.html # Личный кабинет
└── static/            # Статические файлы
    └── style.css      # Стили
```

## Запуск с Docker

### Сборка и запуск контейнера:
```bash
docker-compose up --build
```

### Остановка контейнера:
```bash
docker-compose down
```

Сайт будет доступен по адресу: http://localhost:5000

## Запуск без Docker

### Установка зависимостей:
```bash
pip install -r requirements.txt
```

### Запуск приложения:
```bash
cd website
python app.py
```

## Функционал

- ✅ Главная страница с описанием проекта
- ✅ Форма регистрации пользователей
- ✅ Форма входа
- ✅ Личный кабинет с информацией о пользователе
- ✅ Система сессий и аутентификации

## Примечание

В текущей версии данные пользователей хранятся в памяти. Для production использования рекомендуется подключить базу данных (PostgreSQL, MySQL или SQLite).
